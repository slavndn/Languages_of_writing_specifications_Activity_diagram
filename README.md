# Диаграмма деятельности: подготовка к защите ВКР

## Описание
В репозитории хранится диаграмма деятельности (UML Activity Diagram), отражающая процесс подготовки к защите ВКР по теме:

**«Разработка веб-приложения для учета спортивных достижений обучающихся на уроках физической культуры в школе»**

Диаграмма включает подготовку всех необходимых элементов для защиты: текст ВКР, отзыв руководителя, презентация, рецензия (при необходимости), приложения и финальный пакет документов.
